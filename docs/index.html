【国内消息】
6月17日，中国拟斥资2万亿元推进国家AI算力网络，计划纳入华为、阿里、壁仞、摩尔线程等九类国产AI芯片，全面排除境外算力产品，强力推进国产替代。
6月17日，上述计划属于中国【六网】基础设施大计划组成部分，亦是十四五规划数据基础设施优先建设方向的延伸，融资方式拟使用主权债和超长期国债，若含电网升级全口径投资或超5万亿元。
