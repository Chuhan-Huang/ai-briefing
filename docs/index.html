全球AI算力需求首次覆盖折旧成本，数据中心光互联成新瓶颈，国产算力租赁景气延续
2026-06-30
更新于 2026-06-30 10:05 (北京时间)
☀️全球动态
【国际消息】
6月29日，一份行业研究报告显示，2026年Q1全球AI收入（除中国）达250亿美元，首次连续两个季度超过数据中心与芯片投资折旧成本，但折旧仍消耗收入逾三分之二，盈利空间依然偏窄。
6月29日，分析人士指出，超大规模算力支出持续加码，每十亿美元GPU支出均同步拉动光模块、交换机、光纤等网络互联需求，算力规模已超出铜线传输极限，光互联成为AI数据中心最核心瓶颈。
6月29日，美国AI数据中心基础设施投资商AIB Data Centers宣布纳入罗素微盘股指数，此前完成约6325万美元公开募股，专注AI高性能计算数据中心开发与运营。
6月29日，私募市场研究显示，全球超大规模云厂商为满足算力融资需求，基础设施基金规模持续创新高，私募资本成为数据中心建设的重要补充来源，当前约有695只基础设施基金在募，合计目标规模达5550亿美元。
【国内消息】
6月30日，全球AI算力基础设施投资热潮背景下，国家工信部【算力互联互通行动计划】持续推进，目标2026年底建成完备算力互联互通标准体系，推动跨厂商、跨地区算力协同落地。
6月30日，行业数据显示，国内大型数据中心液冷渗透率已从2025年初的5.3%飙升至近24%，高端训练集群液冷渗透率超70%，政策要求2026年底新建大型数据中心液冷渗透率须达60%以上。
6月30日，国内AI算力租赁行业Q1业绩全面兑现，头部算力租赁企业一季报营收与净利润同比大幅增长，算力需求从政策驱动加速转向真实市场订单兑现。
