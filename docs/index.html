【国际消息】
6月24日，全球领先大模型厂商与芯片巨头联合发布首款专为大语言模型推理场景定制的ASIC加速器，从立项到流片仅历时9个月，创下高性能先进半导体最快ASIC开发纪录；该芯片早期测试显示性价比较现有主流GPU提升约50%，预计2026年底开始在吉瓦级数据中心规模部署。

6月25日，全球AI基础设施加速迈入高功率密度时代：单机柜功耗从传统30-40kW突破至数百kW乃至接近1MW量级，新一代GPU机架系统预计单柜功耗达300kW，液冷已成数据中心必配基础设施而非可选方案。
【国内消息】
6月25日，国内AI数据中心液冷渗透率加速提升：受国家政策强制要求2026年底新建大型数据中心液冷渗透率不低于60%、PUE不超过1.15推动，高端训练集群液冷渗透率已超70%，液冷标准落地年加速来临。
6月25日，国产算力自主可控进程提速：据行业报告，昇腾950PR单卡推理性能已达同类进口产品2.87倍且为国内唯一支持FP4低精度的推理产品，头部互联网厂商2026年Q1国产芯片订单超220亿元，国产替代从政策驱动转向市场化验证。
6月25日，工信部算力互联互通行动计划推进：2026年目标建成完备算力互联互通标准体系，推动算力需求从传统裸金属租赁向Token分成模式升级，国内AI算力服务进入高质量发展新阶段。
